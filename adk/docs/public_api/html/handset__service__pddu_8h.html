void HandsetServicePddu_CheckToImportBtDeviceHandsetServiceConfig ( void  )

Check whether we need to initialise the Handset Service PDD from a previous ADK BT Device PDD.

If the device has been upgraded from a software build based on an ADK version between ADK21.1 and ADK22.2, we shall need to initialise the handset service PDD from the BT Device PDD (where it was stored in those software versions). This function shall therefore create the device_property_hadset_service from the device_property_handset_config.
