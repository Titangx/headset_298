Detailed Description

Map of user PS Keys.

Add a PS Key here if it is going to be used. If there is possibility that a PS Key would need to be populated during deployment then it should also be added to subsys7_psflash.htf.

PS Key used for various charger comms controls. This key can be extended for any debug, test or general configurations are exposed. Current format is: Index [0]: If set, will enable debug-over-charger-comms and disable entering dormant mode when in the charging case. This is only applicable to SchemeB.
