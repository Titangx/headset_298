◆ bredr_scan_manager_scan_type_t

enum bredr_scan_manager_scan_type_t

#include <adk/src/domains/bt/bredr_scan_manager/bredr_scan_manager.h>

Enumeration of types of scanning parameters.

This is an ordered list, the higher the value the higher the priority of the parameters. So a request for slow parameters when fast are already running will not be honoured (though the scan will still 'start'), while a request for fast parameters when slow are already configured will result in the parameters being changed to fast.

Enumerator
SCAN_MAN_PARAMS_TYPE_PENDING_RELEASE 

Indicates a client that has requested release, for which we shouldn't enable scan. It shall be removed once the bredr scan manager recieves indication that the scan has been released.

Slow parameters type, lower duty cycle.

SCAN_MAN_PARAMS_TYPE_FAST 

Fast parameters type, higher duty cycle.

SCAN_MAN_PARAMS_TYPE_THROTTLE 

Throttle parameters type, balanced duty cycle

SCAN_MAN_PARAMS_TYPE_MAX

Total number of types

◆ BredrScanManager_PauseTruncatedPageScan()

void BredrScanManager_PauseTruncatedPageScan ( Task  disabler)

#include <adk/src/domains/bt/bredr_scan_manager/bredr_scan_manager.c>

Disable truncated page scanning.

Parameters
disablerThe task requesting the disable.

Any active truncated page scanning will be paused until BredrScanManager_ResumeTruncatedPageScan is called. The disabler task will be sent a #BREDR_SCAN_MANAGER_PAUSE_TPS_CFM_T when scanning has been paused successfuly (the paused field will be set to TRUE).

If BredrScanManager_ResumeTruncatedPageScan is called before scanning has paused, a #BREDR_SCAN_MANAGER_PAUSE_TPS_CFM_T message will be sent to the disabler task with the disabled field set to FALSE.

Only one client may use this function, it is not intended for general use. Generally, clients should use the BredrScanManager_PageScanRelease or BredrScanManager_InquiryScanRelease to release their request for scanning to be enabled.

◆ BredrScanManager_ScanDisable()

void BredrScanManager_ScanDisable ( Task  disabler)

#include <adk/src/domains/bt/bredr_scan_manager/bredr_scan_manager.c>

Disable page and inquiry scanning.

Parameters
disablerThe task requesting the disable.

Any active scanning will be paused until BredrScanManager_ScanEnable is called. The disabler task will be sent a BREDR_SCAN_MANAGER_DISABLE_CFM_T when scanning has been disabled successfuly (the disabled field will be set to TRUE).

If BredrScanManager_ScanEnable is called before active scanning has paused, a BREDR_SCAN_MANAGER_DISABLE_CFM_T message will be sent to the disabler task with the disabled field set to FALSE.

Only one client may use this function, it is not intended for general use. Generally, clients should use the BredrScanManager_PageScanRelease or BredrScanManager_InquiryScanRelease to release their request for scanning to be enabled.
